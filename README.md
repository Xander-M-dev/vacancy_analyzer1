# Анализатор вакансий с hh.ru

Проект для курсовой работы. Загружает данные о компаниях и вакансиях (из JSON-файла, т.к. API hh.ru возвращает 403), сохраняет в PostgreSQL и предоставляет аналитику через класс DBManager.

## Структура
- main.py – точка входа, создание БД, загрузка данных, меню.
- db_manager.py – класс для SQL-запросов.
- utils.py – создание БД и таблиц.
- config.py – чтение параметров из .env.
- data/vacancies.json – тестовые вакансии (25 записей, 5 компаний).

## Запуск

 `poetry run python main.py`
